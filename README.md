# YAGA
Yet Another Geo Aggregator



## Prerequisites
- Python PEP 8 (for checking code-style in local test)

  ```bash
  pip install pep8
  ```
- Python Py4J == 0.9.0 (for initiating Spark in local test)

  ```bash
  pip install -r requirements.txt
  ```

## Test

- For test in your local, just run below:

    ```
    ./dev/run_tests
    ```
